Import XML Characters from PC Gen. Please follow the instructions below.

In Hero Lab:

  1. Open your character
  2. Select the File Menu
  3. Select ‘Export’
  4. Select ‘csheet_fantasy_generic_export.xml.ftl’
  5. Click on the export button
  6. Select the export file and click save
